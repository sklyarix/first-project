# Git commands

```
git init //Сделать папку c репозиторием

rm -rf .git	 //«Разгитить» папку, если что-то пошло не так

git status	//Проверить состояние репозитория

git add //Подготовить файлы к сохранению

git commit -m  //Выполнить коммит -m добавить сообщение

git log //Просмотреть историю коммитов

git remote add// Пример:  git remote add origin git@github.com:%ИМЯ_АККАУНТА%/first-project.git	Привязать удалённый репозиторий к локальному

git remote -v //Убедиться, что репозитории связаны. -v - Он позволяет показать больше информации в выводе.

git push //  (-u) - если первый	Отправить изменения на удалённый репозиторий

```
